Name

PostGIS_Scripts_Build_Date — Returns build date of the PostGIS scripts.

Synopsis

text PostGIS_Scripts_Build_Date();

Description

Returns build date of the PostGIS scripts.

Availability: 1.0.0RC1

Examples

SELECT PostGIS_Scripts_Build_Date();
